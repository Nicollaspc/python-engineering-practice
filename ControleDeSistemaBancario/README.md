# 🏦 Sistema de Controle Bancário em Python

> Projeto desenvolvido para simular operações básicas de uma conta bancária, com foco em Programação Orientada a Objetos (POO) e boas práticas.

---

## 📌 Sobre o projeto

Este sistema simula o funcionamento de uma conta bancária via terminal, permitindo realizar operações como:

* Depósito
* Saque
* Consulta de saldo
* Histórico de transações

O projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de programação e modelagem de sistemas.

---

## 🧠 Conceitos aplicados

* Programação Orientada a Objetos (POO)
* Encapsulamento
* Uso de propriedades (`@property`)
* Tratamento de exceções (`try/except`)
* Organização de código
* Separação de responsabilidades (lógica vs interface)

---

## ⚙️ Funcionalidades

* Criar uma conta com titular e saldo inicial
* Realizar depósitos com validação
* Realizar saques com verificação de saldo
* Exibir status da conta
* Registrar e exibir histórico de operações

## ▶️ Como executar

1. Clone o repositório:

```bash

```

2. Acesse a pasta:

```bash
cd controle-bancario
```

3. Execute o programa:

```bash
python main.py
```
